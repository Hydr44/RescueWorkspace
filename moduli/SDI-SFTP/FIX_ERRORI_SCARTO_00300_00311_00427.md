# 🔧 Fix Errori Scarto SDI - 00300, 00311, 00427

## ❌ Errori Ricevuti da SDI

### 1. **Errore 00300** - IdCodice non valido
```
1.1.1.2 <IdCodice> non valido : 02166430856
```

**Causa:** 
- L'IdCodice potrebbe contenere spazi, caratteri speciali o il prefisso "IT" che non viene rimosso correttamente
- SDI verifica che l'IdCodice sia registrato nell'anagrafe tributaria

**Soluzione applicata:**
- ✅ Normalizzazione IdCodice: rimozione spazi, caratteri speciali e prefisso "IT"
- ✅ Validazione lunghezza dopo normalizzazione (11 caratteri per P.IVA, 16 per CF)
- ✅ Messaggi di errore più chiari che indicano il valore normalizzato

---

### 2. **Errore 00311** - CodiceDestinatario non valido
```
1.1.4 <CodiceDestinatario> non valido : CodiceDestinatario : MJ1OYN non trovato
```

**Causa:**
- Il CodiceDestinatario "MJ1OYN" (6 caratteri) non è valido nell'anagrafica IPA
- Potrebbe essere un codice non attivo o errato

**Nota:** Questo è un errore di dati, non di codice. Il CodiceDestinatario deve essere valido e attivo nell'anagrafica IPA.

---

### 3. **Errore 00427** - Lunghezza CodiceDestinatario errata
```
1.1.4 <CodiceDestinatario> di 7 caratteri a fronte di 1.1.3 <FormatoTrasmissione> 
con valore FPA12 o 1.1.4 <CodiceDestinatario> di 6 caratteri a fronte di 
1.1.3 <FormatoTrasmissione> con valore FPR12
```

**Causa:**
- Il CodiceDestinatario "MJ1OYN" è di **6 caratteri**
- Il FormatoTrasmissione era hardcoded a **FPR12** (che richiede 7 caratteri)
- **Mismatch:** 6 caratteri con FPR12 = ERRORE

**Soluzione applicata:**
- ✅ **Determinazione automatica FormatoTrasmissione** in base alla lunghezza del CodiceDestinatario:
  - CodiceDestinatario di **6 caratteri** → `FPA12` (fattura verso PA)
  - CodiceDestinatario di **7 caratteri** → `FPR12` (fattura verso privato)
- ✅ Versione XML (`versione` attributo) ora corrisponde al FormatoTrasmissione determinato
- ✅ Normalizzazione CodiceDestinatario alla lunghezza corretta

## ✅ Modifiche Applicate

### 1. Normalizzazione IdCodice (`xml-generator.js`)

```javascript
// PRIMA
const idCodice = cedente.id_fiscale_iva?.id_codice || cedente.id_codice;

// DOPO
let idCodice = cedente.id_fiscale_iva?.id_codice || cedente.id_codice;
// Normalizza: rimuovi spazi, caratteri speciali e prefisso "IT"
idCodice = String(idCodice).trim().replace(/\s+/g, '').replace(/^IT/i, '');
// Validazione lunghezza dopo normalizzazione
if (idCodice.length !== 11 && idCodice.length !== 16) {
  throw new Error(`IdCodice non valido: ... Valore normalizzato: "${idCodice}" ...`);
}
```

### 2. Determinazione Automatica FormatoTrasmissione

```javascript
// PRIMA (hardcoded)
const formatoTrasm = sdi.trasmissione?.formato_trasmissione || 'FPR12';
const versioneAttributo = 'FPR12';

// DOPO (automatico)
const codiceDest = sdi.trasmissione?.codice_destinatario || '';
let formatoTrasm = sdi.trasmissione?.formato_trasmissione;

// Se non specificato, determina dalla lunghezza del CodiceDestinatario
if (!formatoTrasm) {
  const codiceLen = codiceDest.trim().length;
  if (codiceLen === 6) {
    formatoTrasm = 'FPA12'; // Fattura verso PA
  } else if (codiceLen === 7 || codiceLen === 0) {
    formatoTrasm = 'FPR12'; // Fattura verso privato (default)
  } else {
    formatoTrasm = 'FPR12'; // Default
  }
}

const versioneAttributo = formatoTrasm; // Versione XML = FormatoTrasmissione
```

## 📋 Regole SDI

### FormatoTrasmissione e CodiceDestinatario

| FormatoTrasmissione | CodiceDestinatario | Tipo Fattura |
|---------------------|-------------------|--------------|
| **FPA12** | **6 caratteri** | Fattura verso Pubblica Amministrazione |
| **FPR12** | **7 caratteri** | Fattura verso Privato/Azienda |

**Errore 00427:** Si verifica quando c'è un mismatch tra FormatoTrasmissione e lunghezza CodiceDestinatario.

## ⚠️ Note Importanti

### Errore 00300 - IdCodice non valido

Se l'errore persiste dopo la normalizzazione, potrebbe significare:
1. **P.IVA non registrata** nell'anagrafe tributaria SDI
2. **P.IVA errata** nei dati azienda
3. **P.IVA non attiva** o sospesa

**Verifica:**
- Controlla che la P.IVA sia corretta nei Settings
- Verifica che la P.IVA sia registrata e attiva su SDI
- Se necessario, contatta SDI per verificare lo stato della P.IVA

### Errore 00311 - CodiceDestinatario non trovato

Questo è un **errore di dati**, non di codice:
- Il CodiceDestinatario deve essere **valido e attivo** nell'anagrafica IPA
- Per verificare un codice destinatario: https://www.indicepa.gov.it/ricerca
- Se il codice è di 6 caratteri, deve essere una PA valida
- Se il codice è di 7 caratteri, deve essere un'azienda/privato con codice SDI valido
