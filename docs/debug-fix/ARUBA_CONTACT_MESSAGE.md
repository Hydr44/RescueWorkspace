## **Cosa Aspettarsi**

Dopo l'invio, Aruba dovrebbe:
1. **Rispondere entro 24-48h** (solitamente)
2. **Farti domande** su volume fatture, esigenze specifiche
3. **Proporre un piano** (base o Premium)
4. **Fornire documentazione** API se disponibile
5. **Eventualmente organizzare una call** per approfondire
