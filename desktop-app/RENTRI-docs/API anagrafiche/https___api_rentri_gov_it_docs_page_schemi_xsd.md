## <span id="specifiche-tecniche">Specifiche tecniche</span>

### <span id="xsd-di-carattere-generale">XSD di carattere generale</span>

- [rentri-common-1.0.xsd](/docs/assets/rentri-common-1.0.xsd?v=1.0.1)
  *(ver 1.0.1)* - Pattern condivisi per la validazione degli
  identificativi in RENTRI (es. IdentificativoRegistro,
  NumIscrOperatore, NumIscrSito, CodiceFiscale, …).
- [rentri-enum-1.0.xsd](/docs/assets/rentri-enum-1.0.xsd?v=1.0.2) *(ver
  1.0.2)* - Enumerazioni (es. CausaliOperazione, StatiFisici,
  UnitaMisura, …).
- [xmldsig-core-schema.xsd](/docs/assets/xmldsig-core-schema.xsd?r=1.2)
  *(rev 1.2)* - Schema standard W3C per firme digitali XML.

### <span id="xsd-per-la-definizione-dei-registri-di-c-s-e-dei-movimenti-in-formato-digitale">XSD per la definizione dei registri di C/S e dei movimenti in formato digitale</span>

- [rentri-registri-1.0.xsd](/docs/assets/rentri-registri-1.0.xsd?v=1.0.1)
  *(ver 1.0.1)* - Struttura del Registro di carico/scarico tenuto in
  modalità digitale, completo della Vidimazione Virtuale fornita da
  RENTRI.
- [rentri-movimenti-1.0.xsd](/docs/assets/rentri-movimenti-1.0.xsd?v=1.0.3)
  *(ver 1.0.3)* - Dettaglio della struttura dati RENTRI di un movimento
  annotato nel registro di carico/scarico.

In [questa pagina](javascript:loadPage('registro-digitale')) è possibile
trovare la guida tecnica che spiega la struttura del registro di
carico/scarico in formato digitale, con relativa *Vidimazione Virtuale*
fornita da RENTRI, dei movimenti registrati nello stesso e delle
validazioni in fase di trasmissione.
