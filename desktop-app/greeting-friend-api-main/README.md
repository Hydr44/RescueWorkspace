# RescueManager Desktop Application

Sistema completo di gestione trasporti, logistica e demolizioni per l'industria del recupero.

## 🚀 Caratteristiche

- **Gestione Trasporti**: Pianificazione e monitoraggio trasporti con autisti e veicoli
- **Gestione Clienti**: Database clienti con codice univoco per tracciamento
- **Trasporto Veloce**: Creazione rapida trasporti con form semplificato
- **Demolizioni**: Gestione pratiche demolizioni con integrazione RVFU/SDI
- **Fatturazione**: Sistema completo di fatturazione e preventivi
- **Ricambi**: Gestione inventario ricambi auto
- **OAuth Authentication**: Autenticazione sicura con OAuth 2.0
- **Sync Cloud**: Sincronizzazione real-time con piattaforma web
- **Remote Control**: Controllo remoto manutenzione e aggiornamenti
- **Multi-org**: Gestione multi-organizzazione

## 👨‍💻 Sviluppatore

**haxies** - 2025

Sviluppato con React, Electron, TypeScript e Supabase.

## 🛠️ Tecnologie

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Desktop**: Electron
- **Database**: Supabase (PostgreSQL)
- **Local DB**: SQLite (better-sqlite3)
- **Authentication**: OAuth 2.0, Supabase Auth

## 📦 Setup Sviluppo

```bash
# Installa dipendenze
npm install

# Avvia in modalità sviluppo
npm run dev

# Build
npm run build
```
